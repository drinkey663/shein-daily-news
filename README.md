# SHEIN每日热点资讯推送 Skill

自动爬取SHEIN过去24小时的最新动态，重点关注外部合作、投融资、跨境电商等内容，支持RSS源和News API，每日定时推送到钉钉群。

## 功能特点

- 🤖 **自动爬取**：从RSS源和News API自动爬取新闻
- ⏱️ **时间过滤**：只获取过去24小时内的新闻
- 📊 **智能分类**：自动分类为投融资、外部合作、跨境电商、监管动态等
- 🔄 **去重处理**：自动去除重复新闻
- 📱 **钉钉推送**：定时推送到钉钉群机器人
- 📎 **原文链接**：附带原文链接，便于深入了解

### 爬虫流程

```
1. 从多个RSS源获取新闻
2. 从News API获取新闻（如已启用）
3. 过滤：只保留包含SHEIN/希音关键词的新闻
4. 过滤：只保留过去24小时内的新闻
5. 去重：基于标题相似度去重
6. 分类：自动分类新闻类型
7. 格式化：生成钉钉消息格式
8. 推送：发送到钉钉群
```

## 快速开始

### 1. 安装依赖

```bash
pip3 install requests
```

### 2. 配置钉钉机器人

1. 在钉钉群中添加自定义机器人
2. 获取 `access_token` 和 `secret`（如启用加签）
3. 编辑 `scripts/shein_daily_news.py`，修改配置：

```python
ACCESS_TOKEN = "你的机器人access_token"
SECRET = "你的机器人secret"
```

### 3. 测试发送

```bash
python3 scripts/shein_daily_news.py
```

### 4. 设置定时任务

```bash
crontab -e

# 添加以下行（每天上午9:00推送）
0 9 * * * /usr/bin/python3 /path/to/shein_daily_news.py >> /path/to/shein_cron.log 2>&1
```

## 可选：配置News API

如需获取更多国际新闻，可申请News API密钥：

1. 访问 https://newsapi.org/ 注册账号
2. 获取API Key
3. 编辑脚本，启用NewsAPI：

```python
NEWS_APIS = {
    "newsapi": {
        "enabled": True,
        "api_key": "你的NewsAPI密钥"
    }
}
```

## 自定义配置

### 修改推送时间

```bash
crontab -e
# 例如改为每天下午2:30
30 14 * * * /usr/bin/python3 /path/to/shein_daily_news.py >> /path/to/shein_cron.log 2>&1
```

### 添加新的RSS源

编辑脚本中的 `RSS_SOURCES` 变量：

```python
RSS_SOURCES = {
    "new_source": {
        "name": "新来源名称",
        "rss_url": "https://example.com/feed.xml",
        "keywords": ["SHEIN", "希音", "shein"]
    }
}
```

### 修改时间窗口

编辑脚本中的 `TIME_WINDOW_HOURS` 变量：

```python
TIME_WINDOW_HOURS = 24  # 修改为需要的小时数，如12表示只获取12小时内的新闻
```

### 修改最大新闻条数

编辑脚本中的 `MAX_NEWS_COUNT` 变量：

```python
MAX_NEWS_COUNT = 8  # 修改为需要的条数
```

## 注意事项

- 请确保钉钉机器人的IP白名单包含运行脚本的服务器IP
- 如未设置加签，`SECRET` 可留空字符串
- RSS源可能会变动，如无法获取请检查RSS地址是否有效
- News API有请求频率限制，免费版每分钟最多100次请求
- 建议定期检查日志文件排查问题

## 故障排查

### 无法获取新闻

1. 检查RSS源是否可访问：
   ```bash
   curl -I https://rss.sina.com.cn/roll/finance/hot_roll.xml
   ```

2. 检查网络连接是否正常

3. 查看日志文件：
   ```bash
   tail -f /path/to/shein_cron.log
   ```

### 钉钉推送失败

1. 检查access_token是否正确
2. 检查secret是否正确（如启用加签）
3. 检查钉钉机器人IP白名单是否包含服务器IP
